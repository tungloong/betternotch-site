Help and issue reporting

Support

BetterNotch requires macOS 26 or later. Its primary purpose is to let the physical notch disappear into the menu bar. Gradient is the signature effect, Solid Black is the alternative, and each display can be configured independently.

Before you begin

Open System Settings → Menu Bar and turn off “Show menu bar background.” If this option remains enabled, the system menu bar backdrop will interfere with BetterNotch’s visual effect.

If the effect does not appear

  1. Make sure Desktop Effect is enabled.
  2. Choose Gradient or Solid Black. Original means no effect is applied to that display.
  3. If you use multiple displays, confirm that you are configuring the intended display.
  4. Switch to another effect once, or turn Desktop Effect off and back on.
  5. If the problem continues, quit and reopen BetterNotch.

Launch at Login

The Launch at Login switch is in BetterNotch Settings.

If macOS says approval is required, select “Open Login Items Settings,” approve BetterNotch in System Settings, then return to the app. BetterNotch will refresh the current status automatically.

Contact support

If the issue is still unresolved, email longbuild@icloud.com and include as much of the following as possible:

  • Your Mac model and macOS version
  • The currently selected effect
  • Your built-in and external display configuration
  • The steps you took before the problem appeared
  • A screenshot if needed, with personal information redacted first

We will do our best to help you identify the cause.
