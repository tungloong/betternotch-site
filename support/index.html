Support

BetterNotch requires macOS 26 or later and is designed for a MacBook display with a physical notch.

English

macOS 14 and 15 are not supported because they do not provide the “Show menu bar background” control required by the desktop effect.

Before reporting an issue:

  1. In System Settings → Menu Bar, turn off “Show menu bar background.”
  2. Try the same steps with any wallpaper type, including dynamic, aerial, or video wallpapers; the overlay does not replace the wallpaper.
  3. If the result looks stale, choose Original Wallpaper, then enable Gradient or Solid Black again.
  4. If Launch at Login needs approval, open BetterNotch Settings and use “Open Login Items Settings,” then return to BetterNotch so it can refresh the system status.
  5. When reporting a problem, include the Mac model, macOS version, selected effect, connected-display setup, and exact steps to reproduce. Do not attach a private wallpaper unless you are comfortable sharing it.

Support email: longbuild@icloud.com

简体中文

BetterNotch 要求 macOS 26 或更高版本，并面向带物理刘海的 MacBook 显示屏。macOS 14 和 15 不受支持，因为它们没有桌面效果所需的“显示菜单栏背景”开关。

反馈问题前，请先尝试：

  1. 在“系统设置 → 菜单栏”中关闭“显示菜单栏背景”。
  2. 任意壁纸类型都可以使用，包括动态、航拍或视频壁纸；图层不会替换壁纸。
  3. 如果效果没有更新，请先选择“原始壁纸”，再重新选择“渐变”或“纯黑”。
  4. 如果“登录时启动”需要批准，请在 BetterNotch 设置中选择“打开登录项设置”，完成批准后返回 BetterNotch，App 会刷新系统状态。
  5. 反馈时请提供 Mac 型号、macOS 版本、所选效果、外接显示器配置和完整复现步骤。除非你愿意分享，否则不要附上私人壁纸。

支持邮箱：longbuild@icloud.com
